Pertanyaan yang Sering Diajukan

Dynamic Experience adalah platform pendaftaran kegiatan yang menyediakan berbagai event seperti Seminar, Konser, dan Berita. Platform ini memungkinkan Anda untuk mendaftar ke berbagai kegiatan, mengisi daftar hadir, dan mendapatkan sertifikat.

Untuk mendaftar ke sebuah event:

  1. Login ke akun Anda (atau daftar jika belum punya akun)
  2. Buka halaman detail event yang ingin Anda ikuti
  3. Klik tombol "Daftar Sekarang"
  4. Untuk event berbayar, upload bukti pembayaran
  5. Token pendaftaran akan dikirim ke email Anda

Token adalah kode 10 digit yang dikirim ke email Anda setelah mendaftar ke sebuah event. Token digunakan untuk mengisi daftar hadir pada hari H kegiatan.

Cara menggunakan token:

  1. Buka halaman "Riwayat" di menu dropdown profil
  2. Cari event yang sudah Anda daftar
  3. Klik tombol "Hadir" setelah event dimulai
  4. Masukkan token 10 digit yang ada di email
  5. Klik "Konfirmasi Kehadiran"

Sertifikat akan otomatis dibuat setelah Anda mengisi daftar hadir dengan token yang valid. Sertifikat hanya dibuat jika event memiliki template sertifikat.

Cara melihat dan mengunduh sertifikat:

  1. Buka menu dropdown profil
  2. Pilih "Sertifikat"
  3. Klik "Download Sertifikat" pada event yang diinginkan

Untuk event berbayar:

  1. Transfer pembayaran ke nomor rekening yang tertera di detail event atau berita
  2. Upload bukti pembayaran saat mendaftar atau setelah terdaftar
  3. Tunggu validasi dari Event Organizer
  4. Setelah pembayaran disetujui, Anda dapat mengisi daftar hadir
  5. Jika pembayaran ditolak, upload bukti pembayaran baru

Jika token Anda hilang:

  1. Cek email Anda (termasuk folder spam) untuk mencari email token
  2. Jika tidak ditemukan, hubungi Event Organizer event tersebut
  3. Informasi kontak Event Organizer dapat dilihat di halaman detail event
  4. Event Organizer dapat membantu memberikan token baru jika diperlukan

Daftar hadir dapat diisi setelah:

  • Event sudah dimulai (setelah jam kegiatan dimulai)
  • Untuk event berbayar: pembayaran sudah disetujui oleh Event Organizer
  • Anda sudah terdaftar ke event tersebut

Catatan: Anda bisa mengisi daftar hadir meskipun telat, selama event belum selesai atau dibatalkan.

Informasi kontak Event Organizer dapat ditemukan di:

  • Halaman detail event (bagian "Event Organizer")
  • Halaman isi daftar hadir (jika token hilang)
  • Halaman riwayat (link email Event Organizer)

Anda dapat menghubungi Event Organizer melalui email atau telepon yang tertera untuk pertanyaan tentang event, token, atau masalah lainnya.
